坎巴拉太空计划：分享自己造SSTO空天飞机的公式

发表于 2018-08-09 |
字数统计: 1.1k | 阅读时长 ≈ 3


在玩KSP，想造SSTO空天飞机的时候，总是处于一种不断尝试迭代（当然不断尝试迭代没有错）的状态。描述一下大概是：

  1. 先造个雏形
  2. 飞一下
  3. 推力不够，加发动机 -> 2.飞一下
  4. 燃料烧太快，加燃料 -> 2.飞一下
  5. 然后进入3, 4循环

结果发现飞船越造越臃肿，越造越大，part数越来越多，真是非常头疼。因为可能要根据不同的任务，造不同荷载的飞船，比如只运小绿人，荷载也就10-20吨。如果要送橙罐，荷载至少50吨。

澄清一下，我这里所说的荷载是去掉所有燃料，所有发动机，剩下所有结构的总质量

我就想：能不能找到SSTO上，荷载，燃料，发动机数量的一些关系呢。我想，这肯定是有关系的，于是我用一艘小型SSTO的数据作为基准，在excel上面把他们的关系都实现出来，以满足这艘SSTO的数据情况，然后再通过这个公式，来建造更大的SSTO进行验证。经过我反复验证和修改，终于得到下面这一系列的关系。可以顺利的建造出SSTO来。

SSTO公式演示

第一行是荷载，里面也有描述，就是去掉所有燃料和引擎之后，剩余的质量，单位是吨。在这个各自里面输入，你想运送的质量，下面会自动计算出你需要带多少液态燃料，多少氧化剂，放多少个空天引擎，也就是加粗的那3个数字。

灰色背景的部分，是通过不断试验得到的一个参数。大家可以在制造和试飞的过程中，对这个参数进行修正。

怎么修正呢？比如说，你飞进kerbin的低轨道了，还剩下600dv，然后你用这600dv做了一些与空间站对接的事情，然后再进行减速构造好再入大气的亚轨道。OK，这个时候，你可以看下自己的资源，发现自己液态燃料还比氧化剂多特别多。也就是说，你经过这一套操作后，依然有过多的液态燃料，那么就可以相应的，带更多的氧化剂，这样你在真空中，就能有更多的DV可以操作。那么，你可以将表格中的17:11，调整为17:12，这样，你进入轨道后，就会有大于600的dv，可以进行操作了。

当然，这也是只针对你本次飞行任务和飞船质量的。如果下次造了更大的飞船，进入轨道后，可能就没有这么多dv了，说明这个参数可能和飞船的总质量有关。可能需要更复杂的关系来表示。

飞船总质量与引擎数量的比值：12.5，这个的想法是，SSTO需要一个比较大的推重比，才能更好的加速，那么这个推重比总是固定的，所以 ，飞船总质量越大，那么引擎数量也越多。它们应该是一个线性关系。
